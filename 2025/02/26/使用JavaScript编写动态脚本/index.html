什么是JavaScript

以一个按钮为例

我们可以先用HTML赋予它结构

再用CSS装饰

而JavaScript便是用来实现动态行为

JavaScript能做什么

1.把所需值存在变量中。如上把输入值存入name

2.对文本片段进行作用。如上把“Player 1： ” 连接到变量

3.运行代码以响应一些事件。如上click

⭐应用程序编程接口API

现成的代码构建块集，允许开发人员实现原本难以或不可能实现的程序。

浏览器API

1.DOM API 对html和css进行操作

2.Geolocation API 检索地理信息

3.Canvas WebGL API 创建2D3D动画

4.音视频API 使用多媒体

第三方API

【例】Twitter API允许网站上显示最新推文

综上，JavaScript一个非常常见的用途是通过DOM API动态修改HTML和CSS来更新用户界面

运行顺序

自己写的和计算机“翻译”的

服务器端代码与客户端代码

通俗即“后端代码”“前端代码”

动态代码与静态代码

指是否有更新网页显示应用程序的能力

如何在页面添加JavaScript

1.内联

直接写在html属性里

2.内嵌

写在 HTML 文件的 <script>​ 标签内

3.外部

写在独立.js文件中，再在HTML 文件中通过 <script>​ 标签引入

4.模板化（ES6）

将 JavaScript 代码分成多个文件，使用 import​ 和 export​ 来组织代码

事实上不要用 JavaScript 污染 HTML ，在JavaScript 应用的每个按钮上都包含该属性将效率更高。

因此改用addEventListener！

遍历所有按钮并为它们分配处理程序

脚本加载策略

内部

外部

script 元素被放置在文档的头部，所以代码被视为一个模块
